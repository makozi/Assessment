# Assessment

Demo: https://makozi.github.io/Assessment

### Installation

Clone the project

 ```git clone https://github.com/makozi/Assessment.git ```
 
Navigate to the folder

 ```cd Assessment ```
 
Install Dependencies

 ```npm install ```
 
Run the project

 ```ng serve ```
